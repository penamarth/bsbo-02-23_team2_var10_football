# Описание операции returnTicket

**Операция:** `returnTicket(ticketId: string)`

**Ссылки:** Прецедент: Возврат билета

**Предусловия:**
- Билет с `ticketId` существует в системе
- Билет имеет статус `Active`
- До начала матча осталось более 24 часов

**Постусловия:**
- Атрибуту `ticket.status` присвоено значение `Cancelled` (модификация атрибута)
- Атрибуту `seat.isAvailable` присвоено значение `true` (модификация атрибута)
- Создан экземпляр `refundPayment` для возврата средств (создание экземпляра)
- Атрибуту `refundPayment.amount` присвоено значение `ticket.price * 0.9` (модификация атрибута)
- Отправлено уведомление об освобождении места (вызов метода `NotificationService.Notify()`)
