# Описание операции reserveTicket

**Операция:** `reserveTicket(matchId: string, seatId: string, customer: Customer)`

**Ссылки:** Прецедент: Бронирование билета

**Предусловия:**
- Матч с `matchId` существует в системе
- Место с `seatId` существует и доступно
- Данные покупателя `customer` корректны
- Покупатель имеет менее 3 активных бронирований

**Постусловия:**
- Создан экземпляр `reservation` класса `Reservation` (создание экземпляра)
- Экземпляр `reservation` связан с `match` (формирование ассоциации)
- Экземпляр `reservation` связан с `seat` (формирование ассоциации)
- Экземпляр `reservation` связан с `customer` (формирование ассоциации)
- Атрибуту `seat.isAvailable` присвоено значение `false` (модификация атрибута)
- Атрибуту `reservation.expiryDate` присвоено значение `DateTime.Now + 24 hours` (модификация атрибута)
- Отправлено уведомление о бронировании места (вызов метода `NotificationService.Notify()`)
