# Описание операции buySeasonTicket

**Операция:** `buySeasonTicket(season: string, seatId: string, customer: Customer, paymentMethod: PaymentMethod)`

**Ссылки:** Прецедент: Покупка абонемента

**Предусловия:**
- Сезон `season` активен для продажи абонементов
- Место с `seatId` существует и доступно на все матчи сезона
- Данные покупателя `customer` корректны
- В системе есть список всех матчей сезона

**Постусловия:**
- Создан экземпляр `seasonTicket` класса `SeasonTicket` (создание экземпляра)
- Экземпляр `seasonTicket` связан с `seat` (формирование ассоциации)
- Экземпляр `seasonTicket` связан с `customer` (формирование ассоциации)
- Экземпляр `seasonTicket` связан со всеми матчами сезона (формирование ассоциации)
- Место `seat` зарезервировано на все матчи сезона (модификация атрибута)
- Создан экземпляр `payment` класса `Payment` (создание экземпляра)
- Атрибуту `payment.amount` присвоена стоимость абонемента (модификация атрибута)
- Отправлено уведомление о продаже абонемента (вызов метода `NotificationService.Notify()`)
