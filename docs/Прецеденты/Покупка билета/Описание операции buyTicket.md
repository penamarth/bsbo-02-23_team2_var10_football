# Описание операции buyTicket

**Операция:** `buyTicket(matchId: string, seatId: string, customer: Customer, paymentMethod: PaymentMethod)`

**Ссылки:** Прецедент: Покупка билета

**Предусловия:**
- Матч с `matchId` существует в системе
- Место с `seatId` существует и доступно для продажи
- Данные покупателя `customer` корректны

**Постусловия:**
- Создан экземпляр `ticket` класса `Ticket` (создание экземпляра)
- Экземпляр `ticket` связан с `match` (формирование ассоциации)
- Экземпляр `ticket` связан с `seat` (формирование ассоциации)
- Экземпляр `ticket` связан с `customer` (формирование ассоциации)
- Место `seat` помечено как проданное (модификация атрибута `seat.isAvailable = false`)
- Создан экземпляр `payment` класса `Payment` (создание экземпляра)
- Атрибуту `payment.amount` присвоена стоимость билета (модификация атрибута)
- Атрибуту `ticket.status` присвоено значение `Active` (модификация атрибута)
- Отправлено уведомление об изменении доступности места (вызов метода `NotificationService.Notify()`)
