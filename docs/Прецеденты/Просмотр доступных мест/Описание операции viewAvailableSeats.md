# Описание операции viewAvailableSeats

**Операция:** `viewAvailableSeats(matchId: string)`

**Ссылки:** Прецедент: Просмотр доступных мест

**Предусловия:**
- Матч с `matchId` существует в системе
- Схема стадиона загружена

**Постусловия:**
- Получен список всех мест стадиона (запрос к коллекции `stadium.sections.seats`)
- Для каждого места определен статус доступности (обращение к атрибуту `seat.isAvailable`)
- Сформирован список доступных мест (фильтрация коллекции)
- Информация отображена пользователю (вызов метода отображения)
- Статистика рассчитана: общее количество, свободных, проданных, забронированных (агрегация данных)
