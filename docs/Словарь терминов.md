## Определения

| Термин | Определение | Формат | Правило верификации |
|--------|-------------|--------|---------------------|
| Билет | Документ, подтверждающий право на посещение конкретного матча на конкретном месте | Номер билета: MATCH-SECTOR-ROW-SEAT | Уникальность номера в рамках матча |
| Абонемент | Документ, дающий право на посещение всех домашних матчей команды в течение сезона на конкретном месте | Номер абонемента: SEASON-SECTOR-ROW-SEAT | Уникальность номера в рамках сезона |
| Матч | Футбольное соревнование между двумя командами в определенное время | ID матча, дата, время, команды | Дата не может быть в прошлом |
| Место | Конкретное посадочное место на стадионе | Сектор-Ряд-Место (например, A-10-15) | Место должно существовать на стадионе |
| Сектор | Часть стадиона, объединяющая группу мест | Буквенный или числовой идентификатор | - |
| Касса | Точка продажи билетов | Номер кассы | Уникальность номера |
| Бронирование | Временное резервирование места без оплаты | Номер брони, срок действия | Срок не более 24 часов |
| Категория места | Класс места (VIP, обычное, стоячее) | Enum: VIP, Standard, Standing | - |
| Покупатель | Лицо, приобретающее билет | ФИО, контактные данные | Обязательные поля: ФИО, телефон |
